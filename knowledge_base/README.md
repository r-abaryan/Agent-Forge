# Knowledge Base

Add text files (.txt, .md) or JSON files here for RAG retrieval.

## Supported Formats

- **Text files (.txt, .md)**: Plain text documents
- **JSON files (.json)**: Structured documents with metadata

## JSON Format

```json
{
  "title": "Document Title",
  "content": "Document content...",
  "metadata": {
    "category": "Category",
    "tags": ["tag1", "tag2"]
  }
}
```
